O sistema é singular e será vendido para diferentes empresas. Para o funcionamento, existirá o cadastro/login da empresa e, assim que logada, a empresa usufluirá das funcionalidades.

O sistema será guiado a partir de um usuário logado. Existirão 2 tipos de usuários, um administrador do sistema, que cadastra a empresa do cliente, e um usuario referente à empresa cadastrada. O administrador tem poder sobre o sistema e as ações que infringem as empresas, como Cadastrar, Editar, Vizualizar e Excluir, já a empresa, tem poder apenas sobre as funcionalidades do sistema vendido.
